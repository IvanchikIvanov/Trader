# Как передать «какие свечи я вижу как крючки»

Цель: собрать **золотой набор** (gold set) — свечи, которые **ты** считаешь крючками. По ним подкручиваем детектор, а не спорим «в теории».

---

## Самый простой способ (в чат)

Одним сообщением, списком:

```
BTC 15m long (время московское):
- 9 июля 05:15
- 9 июля 12:00
- 10 июля 03:45

BTC 15m short:
- 8 июля 18:30

НЕ крючок (ложняк):
- 10 июля 14:15 long — слишком глубоко
```

### Важно про время

| Вопрос | Как писать |
|--------|------------|
| Какая свеча? | **Свеча-крюк** (красная long / зелёная short), не весь импульс |
| Какое время? | Время **открытия** 15m свечи |
| **Часовой пояс** | **По умолчанию — московское (MSK, UTC+3)** |
| HTML-график | Ось времени в файлах `charts/*.html` — **UTC** (MSK = UTC + 3 часа) |
| Символ | BTC или ETH |

Пример:

> BTCUSDT 15m **long** крюк, open **9 июля 05:15 MSK** (= 02:15 UTC)

Конвертация: `UTC = MSK − 3 часа`.

## Скриншоты (тоже ок)

1. Скрин TradingView / нашего HTML  
2. Стрелка на **свечу крюка**  
3. Подпись: long/short + дата/время  

Я сам переведу в строку CSV.

---

## Формат CSV (если правишь сам)

```csv
symbol,time_utc,side,timeframe,status,note
BTCUSDT,2026-07-09T05:15:00+00:00,long,15m,gold,clean pullback after HH
BTCUSDT,2026-07-10T14:15:00+00:00,long,15m,reject,4 reds / not a hook for me
```

- `gold` — да, это крючок, детектор **должен** ловить  
- `reject` — похоже, но **не** брать (антипример)

---

## Как мы это используем

1. Ты шлёшь 10–30 gold + несколько reject  
2. Сверяем: детектор поймал / пропустил / ложняк  
3. Крутим правила (HH lookback, 1–3 red, HTF bias…)  
4. Перегоняем бэктест + график  

Команда сверки (после накопления меток):

```bash
$env:PYTHONPATH = "src"
python -m trader labels --file labels/hooks_gold.csv
```

## Мини-чеклист «это long-крючок для меня»

- [ ] Был импульс вверх / новый хай  
- [ ] 1–3 красные  
- [ ] Эта свеча — **последняя** красная отката (или единственная)  
- [ ] Вход имел бы смысл **на close** этой свечи  

То же зеркально для short (зелёные, после low).
